# Skill: Azure Storage

This skill teaches the Kusto AI Assistant about Azure Storage telemetry.

## What's included

| File | What it does |
|------|-------------|
| `context.md` | Tells the assistant about Storage tables, columns, and data patterns |
| `sample-questions.md` | Example questions you can ask |

## Sample questions you can ask

- "What are the top storage accounts by capacity?"
- "Show me daily transaction trends for the last 2 weeks"
- "Which subscriptions are flagged as critical customers?"
- "Are there any anonymous access requests in the last 24 hours?"
